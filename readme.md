## Rodando localmente

Clone o projeto

```bash
  git clone https://github.com/pedromotta462/Residencia_cinq-monix.git
```

Entre no diretório do projeto

```bash
  cd Residencia_cinq-monix
```

Instale as dependências

```bash
  npm install
```

Inicie o servidor

```bash
  npm run dev
```
